软件特点是可以切分法规条款，并能够识别并自动分类各条款的章节号，无需安装，复制到您的电脑上即可使用，使用步骤与技巧如下：

1)将法规全文复制到"法规全文.txt"文件中，使用组合键Ctrl+S保存，不用关闭文件，以省却再次打开时间
2)双击'法条切刀.exe'后等待约8秒钟
3)解析成功后将在同一文件目录下生成一个excel文件，文件名形如XLSX_2020-02-06-H18M16.xlsx。文件名都包含了年月日、小时和分钟信息，便于文件管理参考。
4)软件将检查章节标题长度，超长则会报警，在同一目录下生成warning.txt，包含形如以下内容：
	警告Warning！标题长度过长！问题标题为：
	7.1 液氯气瓶的贮存安全7.1.1 气瓶不应露天存放，也不应使用易燃、可燃材料搭设的棚架存放，应贮存在专用库房内。
原因是该软件以'行'为基础进行切分计算，偶尔会发生原始文本没有清楚分行的情况，导致对标题的识别失效，所以加入此功能作为提示。


【注意事项】
■暂时只能解析具备以下结构特征的法规文本，具备该文本结构的文件多为正规的法律法规：
	-章、节、条
	-章、条
	-条
■暂时只能解析具备以下结构特征的规范性文件文本：
	-一层文件示例：一、二、
	-二层文件示例：一、（一），（二），（三），二、
■可执行文件'法条切刀.exe'和"法规全文.txt"必须位于同一目录下，且不能更改"法规全文.txt"的文件名。
